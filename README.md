
# 🔍 Discord Messages Scrapper

> A simple tool made for **OSINT** (Open Source Intelligence) purposes — scrape and collect messages from Discord channels.

---

## ⚠️ Disclaimer

This tool is intended for **educational and OSINT research purposes only**.  
Using selfbots violates [Discord's Terms of Service](https://discord.com/terms). Use at your own risk. The author is not responsible for any misuse or account bans.

---

## 📦 Installation

Make sure you have [Node.js](https://nodejs.org/) installed, then run:

```bash
npm install discord.js-selfbot-v13
npm install fs
npm install readline
```

---

## 📌 Requirements

| Dependency | Version |
|---|---|
| Node.js | >= 16.x |
| discord.js-selfbot-v13 | latest |
| fs | built-in |
| readline | built-in |
